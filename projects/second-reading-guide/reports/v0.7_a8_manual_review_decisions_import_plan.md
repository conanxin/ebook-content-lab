# v0.7-A8 Manual Review Decisions Import Plan

## Goal

v0.7-A8 adds a safe importer for human-filled manual review decision CSV files.

The importer is intended for a future step after a human fills a copy of the A7 decisions template.

## Why A8 Only Dry-Runs The Empty Template

The current A7 decisions template is intentionally blank.

A8 must not treat blank results as passing results, and must not fabricate review decisions. Therefore this phase only dry-runs the blank template and expects zero updates.

## Inputs

A8 reads:

- `projects/second-reading-guide/reports/manual_review_tasks_v0.7_a4.csv`
- `projects/second-reading-guide/reports/manual_review_summary_v0.7_a4.json`
- `projects/second-reading-guide/reports/manual_review_decisions_template_v0.7_a7.csv`

A8 does not read private book body material.

## Outputs

A8 writes:

- `projects/second-reading-guide/reports/manual_review_decisions_import_dry_run_v0.7_a8.md`
- `projects/second-reading-guide/reports/manual_review_decisions_import_validation_v0.7_a8.md`
- `projects/second-reading-guide/reports/manual_review_decisions_import_usage_v0.7_a8.md`
- `projects/second-reading-guide/reports/v0.7_a8_manual_review_decisions_import_plan.md`

## Import Safety

The importer defaults to dry-run mode.

Apply mode requires:

- `--apply`
- an exact confirmation string: `IMPORT MANUAL REVIEW DECISIONS INTO second-reading-guide`

Dry-run mode never writes to the authoritative A4 CSV.

## Overwrite Protection

The importer refuses to overwrite an existing nonblank `manual_result` unless `--allow-overwrite` is explicitly set.

A8 does not use overwrite mode.

## Legal Result Rules

Allowed results:

- `pass`
- `needs_fix`
- `blocked`
- `deferred`

`needs_fix`, `blocked`, and `deferred` require notes.

Blank values are ignored and are not treated as pass.

## Relation To A6, A7, And A5

- A6 provides single-task updates.
- A7 provides review packets and a blank decisions template.
- A8 imports a human-filled decisions CSV in bulk.
- A5 promotion preflight remains blocked until imported or individually applied results satisfy the readiness rules.

## Suggested A9

A9 should use a human-filled copy of the decisions template, run the importer in dry-run mode, then apply only after the dry-run report is reviewed.
