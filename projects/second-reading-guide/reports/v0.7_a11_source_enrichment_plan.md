# v0.7-A11 Source Enrichment Plan

## User Feedback Summary

The public preview now works, but it still needs to feel more like a real personal reading page.

Requested improvements:

- Add more information from the book.
- Add original excerpts or source clues.
- Compare the past travel route with today's places.
- Make reading-question answers more specific.
- Keep the envelope-style page but make the expanded sections richer.

## A11 Goal

A11 turns the reading guide into a source-informed public preview.

It remains a draft, but each letter card should now include richer reading guidance: source-informed summary, short original clues, scene notes, route-then, route-now, and then/now comparison.

## Copyright and Personal Reading Note

The user has stated that copyright handling is their responsibility and that this is a personal reading page.

A11 therefore allows curated source-derived guide material, but still avoids publishing source book files or turning the page into a full-text reader.

## Original Excerpt Strategy

The page uses short original clues and explanatory notes.

The goal is to help the reader locate scenes and reading angles, not to reproduce chapters wholesale.

Each excerpt is paired with a note explaining why it matters.

## Private Boundary

A11 may read local source-derived materials to build summaries, but it does not publish local paths, source-layer file names, or the original source files themselves.

Public JSON and web mirror files must remain free of local paths and source-layer filenames.

## Past Travel and Present Places

Each letter receives:

- `route_then`: the travel route or place order as represented by title and source clues.
- `route_now`: a current reading note for the place.
- `then_now_comparison`: a cautious comparison between travel experience then and today's place-reading context.

Where public source coverage is insufficient, the item is marked `needs_source_review`.

## Source Coverage Rule

Only places with a named public source and URL are marked `public_source`.

All other places remain `needs_source_review`; no unsupported current-place fact is asserted.

## Page UI

The envelope cards gain richer expanded sections:

- this letter's source-informed summary
- original excerpts and reading clues
- original scene notes
- past route
- today-place comparison
- expanded reading focus
- expanded answer hint

The page also adds:

- `昔日旅程与今日景点`
- `原文摘录与阅读线索`

Promotion preflight should continue to block reviewed-draft promotion because manual review is incomplete.
