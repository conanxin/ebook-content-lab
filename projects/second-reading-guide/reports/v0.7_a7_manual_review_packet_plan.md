# v0.7-A7 Manual Review Packet Plan

## Goal

v0.7-A7 creates safe manual review packets for the `second-reading-guide` project.

The packets help a human review A4 tasks without modifying the authoritative A4 CSV or changing public reading-guide data.

## Why A7 Does Not Auto-Fill Results

Manual review results require human judgment.

A7 only prepares packets and a blank decisions template. It does not mark any task as passed and does not promote project status.

## Inputs

A7 reads:

- `projects/second-reading-guide/reports/manual_review_tasks_v0.7_a4.csv`
- `projects/second-reading-guide/reports/manual_review_summary_v0.7_a4.json`
- `projects/second-reading-guide/public/book_overview.json`
- `projects/second-reading-guide/public/chapter_reading_cards.json`
- `projects/second-reading-guide/public/key_concepts.json`
- `projects/second-reading-guide/public/quote_index.json`
- `projects/second-reading-guide/public/reading_questions.json`
- `projects/second-reading-guide/reports/manual_review_progress_v0.7_a6.md`
- `projects/second-reading-guide/reports/promote_status_preflight_v0.7_a5.md`

A7 does not read private source material or book body text.

## Outputs

A7 writes:

- `projects/second-reading-guide/reports/manual_review_packet_p0_v0.7_a7.md`
- `projects/second-reading-guide/reports/manual_review_packet_all_v0.7_a7.md`
- `projects/second-reading-guide/reports/manual_review_decisions_template_v0.7_a7.csv`
- `projects/second-reading-guide/reports/manual_review_packet_validation_v0.7_a7.md`
- `projects/second-reading-guide/reports/v0.7_a7_manual_review_packet_plan.md`

## P0 Packet

The P0 packet collects priority tasks that should be reviewed first.

It includes task metadata, current blank result state, structural public context, and a command template for applying a real human decision later.

## Full Packet

The full packet groups all 95 tasks by priority and category.

It is intended for systematic review after P0 tasks are understood.

## Decisions Template

The decisions template is a blank CSV for later human use.

It is not authoritative and does not replace `manual_review_tasks_v0.7_a4.csv`.

All `manual_result` and `notes` cells are intentionally blank.

## Relation To A6

After reviewing a task, use `scripts/update_manual_review_result.py` from A6 to apply that single task result to the authoritative CSV.

Do not batch-fill all results.

## Re-Running Promotion Preflight

After real manual review updates are applied, rerun:

- `scripts/check_manual_review_progress.py`
- `scripts/promote_status.py --dry-run`
- `scripts/check_promote_status.py`

Promotion remains blocked until the A5 readiness rules pass.

## Suggested A8

A8 should either perform real human review updates task by task, or add an import validator for a human-completed decisions CSV.
